hexo的一些常用操作

hexo常用命令

  • hexo n（new）”文章名字” == 创建新文章
  • hexo g（generate） == 生成本地文件
  • hexo s（server） == 启动服务以便本地预览
  • hexo d（deploy） == 把本地文件部署到服务器
  • hexo clean == 清除缓存
  • hexo new page “name” == 新建页面
